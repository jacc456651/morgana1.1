# equity-research — Contexto del módulo

## Qué hace
Genera reportes de investigación de renta variable al estilo institucional (Goldman, JPMorgan, Morgan Stanley).
Cubre el ciclo completo: idea → cobertura inicial → seguimiento earnings → actualización de modelo → tesis.

## Comandos disponibles
| Comando | Descripción |
|---------|-------------|
| `/initiate` | Reporte de cobertura inicial para una empresa |
| `/earnings` | Análisis de resultados trimestrales (earnings update) |
| `/earnings-preview` | Preview pre-earnings con escenarios alcista/bajista |
| `/model-update` | Actualizar modelo financiero con nuevos datos |
| `/thesis` | Crear o actualizar tesis de inversión |
| `/sector` | Reporte de overview sectorial |
| `/screen` | Screener de ideas de inversión |
| `/morning-note` | Nota de apertura de sesión (morning meeting) |
| `/catalysts` | Calendario de catalizadores próximos |

## Skills que usa
