# wealth-management — Contexto del módulo

## Qué hace
Herramientas para asesores de patrimonio: reportes de performance para clientes,
planificación financiera, rebalanceo de cartera, propuestas de inversión y tax-loss harvesting.

## Comandos disponibles
| Comando | Descripción |
|---------|-------------|
| `/client-report` | Generar reporte de performance para un cliente |
| `/client-review` | Preparar reunión de revisión con un cliente |
| `/financial-plan` | Construir o actualizar plan financiero personal |
| `/proposal` | Crear propuesta de inversión para un prospecto |
| `/rebalance` | Analizar drift y generar operaciones de rebalanceo |
| `/tlh` | Identificar oportunidades de tax-loss harvesting |

## Skills que usa
`client-report` · `client-review` · `financial-plan` · `investment-proposal`
`portfolio-rebalance` · `tax-loss-harvesting`
