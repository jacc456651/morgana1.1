# financial-analysis — Modelos Cuantitativos MORGANA

## Qué hace
Valoración institucional y análisis financiero profundo para decisiones buy-side:
DCF, comps, LBO, modelo 3 estados, competitive landscape y auditoría de modelos.

## Comandos disponibles
| Comando MORGANA | Descripción |
|-----------------|-------------|
| `/analiza [TICKER]` | DCF + comps integrados como parte del análisis completo |
| `/modelo [TICKER]` | Modelo integrado 3 estados con proyecciones y scenarios |
| `/audit-modelo` | Auditar modelo Excel: fórmulas, hardcodes, circular refs, BS balance |
| `/lbo [TICKER]` | Modelo LBO: sources & uses + deuda + retornos IRR/MOIC |
| `/competitive-analysis [TICKER]` | Competitive analysis: moat + posicionamiento + 2×2 matrix |
