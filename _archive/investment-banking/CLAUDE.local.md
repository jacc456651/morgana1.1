# investment-banking — Contexto del módulo

## Qué hace
Soporta procesos de M&A sell-side y buy-side: materiales de marketing (teaser, CIM),
modelos de fusión, listas de compradores, cartas de proceso y seguimiento de pipeline.

## Comandos disponibles
| Comando | Descripción |
|---------|-------------|
| `/teaser` | Redactar teaser anónimo de una página |
| `/cim` | Redactar Confidential Information Memorandum completo |
| `/buyer-list` | Construir universo de compradores para proceso sell-side |
| `/merger-model` | Modelo de acreción/dilución para una fusión |
| `/one-pager` | Strip profile de una página con plantilla PPT corporativa |
| `/process-letter` | Redactar carta de proceso o bid instructions |
| `/deal-tracker` | Revisar y actualizar el pipeline de deals activos |

## Skills que usa
`teaser` · `cim-builder` · `buyer-list` · `merger-model` · `strip-profile`
`pitch-deck` · `process-letter` · `deal-tracker` · `datapack-builder`
